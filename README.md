<h1>Late Delivery Prediction Model</h1>


<h2>Description</h2>
Developed and compared four classification models (XGBoost, LightGBM, Random Forest, Decision Tree) to predict parcel late delivery, achieving an average accuracy of 85.19%. This model enables proactive customer communication for potential delays and provides insights for optimising logistics routes.
